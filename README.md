AI Travel Itinerary Generator

An intelligent travel planner which generate personal day wise travel plans based on user input like destination, number of days and interests.

Features
User can input destination, number of days and interests
It will generate day wise travel plan automatically
Activities suggestion based on interests like adventure, food, culture, relax
Simple and fast UI built using Streamlit
Code is modular so easy to update and scale later
Tech Stack
Python
Streamlit
NumPy (used conceptually)
Modular programming approach
How It Works
User gives input:
Destination
Number of days
Interests (comma separated)
System will:
Match interests with predefined activity data
Randomly pick suitable activities
Create a structured itinerary for each day
Project Structure

The project is divided into multiple parts for better understanding and easy maintenance.
Main planner logic is separated from data part so that it can be modified easily without affecting whole system.
