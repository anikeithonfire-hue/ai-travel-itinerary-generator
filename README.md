# 🌍 AI Travel Itinerary Generator

An intelligent travel planner that generates personalized day-wise itineraries based on user preferences such as destination, duration, and interests.

---

## ✨ Features

- 📍 Input destination, number of days, and interests
- 🧠 Generates dynamic day-wise travel plans
- 🎯 Interest-based activity recommendations (adventure, food, culture, relax)
- ⚡ Fast and simple UI using Streamlit
- 🧩 Modular code structure for scalability

---

## 🛠️ Tech Stack

- Python
- Streamlit
- NumPy (conceptual usage)
- Modular Programming (planner + data separation)

---

## 🧠 How It Works

1. User provides:
   - Destination
   - Number of days
   - Interests (comma-separated)

2. System:
   - Maps interests to predefined activity datasets
   - Randomly selects relevant activities
   - Generates structured day-wise itinerary
